遅ればせながらあけましておめでとうございます。2026年もよろしくお願いいたします。

![IMG_4397](../../../assets/img/post/2026-01-03-new-year/IMG_4397.webp)

写真は羽田沖で撮影した今年の初日の出です。今年の元旦は雲が多く、7時すぎにようやく太陽が姿を表してくれました。

<!--more-->

# 去年の反省点

去年は振り返り記事を書いていなかったので、ここで少し振り返りたいと思います。

去年は、YotioSoft としては

- 個人ホームページ yotio.jp の公開
- Rust で Windows のプロセス情報を取得する「winprocinfo」の公開
- Rust 製自作画像処理ツール「rusimg」の公開
- dptran の開発・更新

…と、近年にしてはそれなりに YotioSoft としての活動を続けられていたのではないかなと思います。

個人的なイベントとしても

- 修論発表
- 修士課程修了
- 就職
- 研修・発表
- 旅行

と、そこそこ忙しい年ではありました。特に就職という大きなイベントがあり、周囲の環境は全部変わりました。幸い、職場の人間関係に大変恵まれていますし、（社名はここでは出せませんが）世界最先端の技術と競争力を持ち、なおかつ誰もが知るレベルの大企業なので、誇りを持って楽しく仕事に取り組むことができています。

ただ、去年の前半は良かったものの、後半が惰性で生きてしまったなぁという反省点はあります。個人ホームページの公開、自作ツールの新規公開…も全て前半ですし、修了にしろ就職にしろすべて春以前の出来事です。リアルの方のイベントで新しいことを始めるのは時機の関係もありますが、せめて YotioSoft はじめネット上の活動はもっと新しいことをしていきたいですね。

あとは趣味の開発の方も、今年も相変わらず Rust に Rust、CLI ツールに CLI ツール…と偏ってしまっています。たまには OpenSiv3D や HSP3 に回帰して、GUI アプリをもっと書いていきたいものです。実は去年の暮に開発環境を整えていたので、そろそろいい加減 Cities Box などの開発も再開していきたいなぁと考えています。

# 今年実現したいこと（yotio として）

- 個人ホームページのコンテンツを拡充する
- 引き続き現存の開発を続ける
- mprotect-rs を公開する
- Rust 以外の言語にも手を出してみる
- OpenSiv3D などによる GUI アプリ開発に回帰する
- Yapps の拡充・収益化する

こんなところですかね。個人ホームページ（yotio.jp）の方はあまりコンテンツを拡充できていないので、もっとプログラミング以外の趣味のページを作っていきたいですね。

現存の開発、mprotect-rs、まあこのへんは Rust 開発の話です。ただ偏りすぎないよう、Rust 以外の言語も始めてみたりとか、Cities Box など GUI アプリ開発も再開していきたいなと思っています。

Yapps に関しては、月間 1.5 万 PV くらいになってきたので、そろそろ広告を貼らせていただきたいなと思っています（嫌だったら広告ブロックを使っていただいて構いません）。あとは Claude などを用いた Web アプリ開発の自動化も軌道に乗っているので、さらにアプリを拡充させていきたいなと考えています。

# 今年実現したいこと（個人）

- 一人暮らしを始める
- 何らかの新しい日課を始める
- 美容院デビューする
- もっとイベントに参加する
- 仕事面での向上

一人暮らしは本当は去年中に始めたいと思っていたのですが、あまりの東京近辺の家賃高騰に結局出られずにいました。今の実家からでも通えなくはないんですが、遠目の郊外なので職場までかなり遠いというのと、いい加減自分で生活できるようになりたいというのもあり、今年こそ一人暮らしを始めたいと思っています。

仕事面に関しては社外秘の情報も含むのであまり詳しくは書けませんが、プロジェクトリーダーの仕事が任せられるようなレベルになりたいと思っています。

# おわりに

去年は社会人になり、周囲の環境も何から何まで変わり…と大変な一年でしたが、同時に充実感のある一年でした。今年も引き続き、リアルでの生活とネット上の活動を両立できればと思っています（実を言うと、大学院時代より社会人のほうが時間にも余裕があります…）。

今年は YotioSoft 開設 12 周年です。12年目も変わらず活動を続けていきますので、よろしくお願いいたします。
